Filtering

Common initialization

{% highlight js %} $("#container").chaos({ padding: 5 }); {% endhighlight %}

jQuery Chaos organize method can animate a subset off elements, so to achieve filtering we will hide the other elements before the animation

CSS class that will hide the elements

{% highlight css %} .atom.hide { opacity: 0; } {% endhighlight %}

Code for each filter we want to build

{% highlight js %} $("#container").chaos().organize({ selector: ".blue", beforeAnimation: function() { $(".atom").addClass("hide"); $(".blue").removeClass("hide"); } }); {% endhighlight %}

Before the basic organize functions we will remove the class 'hide', eliminating the filter

{% highlight js %} $(".atom").removeClass("hide"); $("#container").chaos().original(); {% endhighlight %}
