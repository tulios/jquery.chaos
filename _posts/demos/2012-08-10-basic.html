Basic

Common initialization

{% highlight js %} $("#container").chaos({ padding: 5 }); {% endhighlight %}

Using organize and order methods

{% highlight js %} $("#container").chaos().original(); // Default button $("#container").chaos().organize(); // Organize button {% endhighlight %}

Organize also accepts an order param

{% highlight js %} $("#container").chaos().organize({order: "reverse"}); // Organize reverse button {% endhighlight %}
